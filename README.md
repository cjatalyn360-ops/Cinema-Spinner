# Cinema Spinner

## Description
Cinema Spinner is a web application that helps users discover random movies based on selected filters such as genre, year range, and actor. It uses The Movie Database (TMDB) API to fetch real movie data and display recommendations dynamically.

## Features
- Random movie generator with filters (genre, year, actor)
- "Spin Again" functionality for quick re-selection
- Prevents duplicate movie recommendations
- Stores previously viewed movies using localStorage
- Displays movie poster, rating, description, and release year

## Languages & Tools Used
- HTML
- CSS
- JavaScript
- TMDB REST API

## How to Run the Project
1. Download or clone the repository
2. Open the folder in VS Code
3. Install the Live Server extension
4. Right-click `index.html` and select "Open with Live Server"

## Important Note
This project requires a TMDB API key.
You will need to replace the API key in `script.js` with your own.
Some instructions here:
1. Create a free API key at https://www.themoviedb.org/
2. Replace API_KEY in script.js
